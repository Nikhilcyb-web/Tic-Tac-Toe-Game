# 🎮 Tic Tac Toe Game

A responsive Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**. The game allows two players to compete on the same device with automatic win detection, draw detection, and game restart functionality.

---

## 📖 Overview

This project recreates the classic Tic Tac Toe game using vanilla JavaScript. It focuses on DOM manipulation, event handling, game logic, and responsive UI design without using any external libraries.

---

## ✨ Features

- 🎯 Two-player gameplay
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Restart game option
- 📱 Responsive design
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
